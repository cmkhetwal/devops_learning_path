# Python for DevOps & Cloud - 90 Day Learning Path

## Who Is This For?
- Complete beginners with ZERO programming knowledge
- Goal: Build Python skills specifically for DevOps & Cloud Engineering

## How to Use This Program
1. Spend **1-2 hours daily** on each day's lesson
2. Each day has: **Lesson (READ)** → **Exercise (DO)** → **Check (TEST)**
3. Run `python3 tracker.py` daily to log progress and see your stats
4. Take the weekly quiz every Sunday: `python3 quiz.py <week_number>`
5. Don't skip days — each lesson builds on the previous one

## Progress Rating System
- Each exercise auto-checks your solution
- Weekly quizzes score you out of 100
- Tracker shows: completion %, streak, weak areas

## Quick Start
```bash
cd /home/cmk/python/devops-python-path

# Check your progress anytime
python3 tracker.py

# Start Day 1
cat week_01/day_1/lesson.md
python3 week_01/day_1/exercise.py

# Check your exercise
python3 week_01/day_1/check.py

# Take weekly quiz (every Sunday)
python3 quiz.py 1
```
