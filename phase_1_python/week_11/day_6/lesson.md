# Week 11, Day 6: Practice Day - CI/CD & Infrastructure as Code

## Mini-Projects

Today you will build 5 mini-projects that combine everything from Week 11.
Each project uses concepts from Days 1-5: Git analysis, Jenkins API,
GitHub Actions, Jinja2-style templates, and Ansible generation.

### Project 1: Git Repository Analyzer
Analyze a simulated repository and produce a status report including
branch health, commit frequency, and contributor statistics.

### Project 2: CI/CD Pipeline Generator
Given application requirements, generate a complete CI/CD pipeline
configuration with build, test, and deploy stages.

### Project 3: Config Template Engine
Build a template engine that takes application metadata and generates
all necessary configuration files (nginx, Dockerfile, k8s, etc.).

### Project 4: Deployment Automation Script
Create a deployment automation system that coordinates pre-deploy
checks, deployment execution, and post-deploy verification.

### Project 5: Infrastructure Manifest Generator
Generate a complete set of infrastructure manifests for deploying
a microservices application.

---

## Tips

- Reuse patterns from Days 1-5
- Focus on clean, well-structured output
- Think about what a real DevOps engineer would need
- Test each function independently before combining
