## Tips Before You Start

### Tip 1: Plan Before You Code
Read each task completely before writing code. Think about:
- What kind of loop do I need? (for or while)
- What conditions do I need to check?
- Do I need break or continue?

### Tip 2: Start Simple
Get the basic version working first, then add details.

### Tip 3: Test As You Go
Run your code after each task to make sure it works before moving on.

## Mini-Project Summaries

### Project 1: Number Guessing Game
A simulated game where the computer picks a number and the "player" tries to
guess it. You will use a **while loop** with **if/elif/else** to give hints.

### Project 2: Server Health Monitor
A monitoring system that checks multiple servers and reports their status.
Uses **for loops** with **if/elif/else** and **counters**.

### Project 3: Simple Menu System
A menu-driven program that keeps showing options until the user chooses to exit.
Uses a **while loop** with **if/elif** for menu choices.

### Project 4: Password Validator
A program that checks if a password meets complexity requirements.
Uses **for loops** to scan through characters with **conditions** and **counters**.

### Project 5: Port Scanner Simulator
A simulated port scanner that checks servers across port ranges.
Uses **nested loops** with **break** and **continue**.
